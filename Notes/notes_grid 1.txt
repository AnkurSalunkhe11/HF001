 Setup 
1. Download selenium server - 4.15.0 jar file in the s/m  // place it in c: or c drive 


2.goto C:\SELENIUM SERVER STANDALONE 4.34 location --> type cmd in nav bar and  Run cmd to start the selenium grid (cmd prompt)
java -jar selenium-server-4.41.0.jar standalone

java -jar selenium-server-4.34.0.jar standalone

3. URL http://localhost:4444/

(4444 is default port no)

Hub and node setup (multiple m/c) using VM
1. Download jar file in every m/c
2. hub m/c : java -jar selenium-server-4.15.0.jar hub
3. Node m/c:
java -jar selenium-sever-4.15.0.jar node --hub http://<hub-ip>:4444
4. url to see sessions: http://localhost:4444/
